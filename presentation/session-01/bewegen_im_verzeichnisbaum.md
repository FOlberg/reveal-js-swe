### Immer in Bewegung bleiben

die drei wichtigsten Kommandos für alles!

```bash
cd     # change dir
ls     # list dir
pwd    # print working dir
```

Sehr wichtig!!!

```bash
ls -l
ls -a
ls -la
ls --help # für alles weitere ;)
```

`--help` funktioniert bei den allermeisten Kommandos und ist eine gute erste Hilfe!
